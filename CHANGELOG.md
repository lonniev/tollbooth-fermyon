# Changelog

All notable changes to this project are documented here. The format follows
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and the project aims
to adhere to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-07-03

Initial proof-of-concept scaffolding.

### Added
- `crypto/` — Rust WASI component (`dpyc:crypto/ops`) providing secp256k1 ECDH,
  BIP-340 schnorr verify, and AES-256-CBC/GCM. Validated byte-for-byte against
  vectors generated by the `tollbooth-dpyc` wheel, and against a real
  Authority-published bootstrap event.
- `bridge/` — Cloudflare Worker that fetches an operator's kind-30078 bootstrap
  config from WebSocket-only Nostr relays over HTTPS, for Wasm cold start.
- `component/` — Python WASI component scaffolding plus a proven
  `WasiHttpTransport` routing `httpx` over `wasi:http` (no `ssl` module).
- Polyglot CI (Rust, Python, bridge Worker, secret scan), CHANGELOG-driven
  releases, Apache-2.0 license, and ecosystem README.
